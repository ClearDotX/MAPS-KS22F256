Macro Definition Documentation

#define KHCICFG_THSLD_DELAY   0x65 
          In general the worst case transaction is an IN token followed by a data packet from the target followed

by the response from the host. The actual time required is a function of the maximum packet size on the bus. Set the KHCICFG_THSLD_DELAY to 0x65 can meet the worst case.
