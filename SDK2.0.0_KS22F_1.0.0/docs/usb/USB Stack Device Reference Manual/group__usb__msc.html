usb_device_class_config_struct_t *  config,
class_handle_t *  handle 
)

This function is used to initialize the msc class.

Parameters
controllerIdThe controller id of the USB IP. Please refer to the enumeration usb_controller_index_t.
configThe class configuration information.
handleIt is out parameter, is used to return pointer of the msc class handle to the caller.
Returns
A USB error code or kStatus_USB_Success.
