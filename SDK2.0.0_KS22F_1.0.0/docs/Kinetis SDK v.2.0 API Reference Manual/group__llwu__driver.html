LLWU Peripheral driver
Low-Leakage Wakeup Unit (LLWU)

This section describes the programming interface of the LLWU Peripheral driver. The LLWU driver configures the LLWU module to support other IPs.

Function Groups

The LLWU drivers provides these functions:

External wakeup pins configurations

Configure the external wakeup pins' working modes, get and clear the wake pin flags. External wakeup pins are accessed by pinIndex which is started from 1. Numbers of external pins dependes on soc configuation.

Internal wakeup modules configurations

Enable/disable the internal wakeup modules, get and the modules flags. Internal modules are accessed by moduleIndex which is started from 1. Numbers of external pins dependes on soc configuation.

Digital pin filter for external wakeup pin configurations

Configure the digital pin filter of external wakeup pins' working modes, get and clear the pin filter flags. Digital pins filters are accessed by filterIndex which is started from 1. Numbers of external pins dependes on soc configuation.

Digital pin filter for reset pin configuration

Configure the pin filter mode of external reset pin.

Internal module DMA requests configurations

Enable/disable the internal module DMA requrest wakeup.
