PMC Peripheral driver
Power Management Controller (PMC)

This section describes the programming interface of the PMC Peripheral driver. The PMC driver configures the PMC module to support the power management of soc.

Function Groups

The PMC drivers provides these functions:

Low voltage detect configurations

Configure the low voltage detect setting, get and clear the low voltage detect flag status.

Low voltage warning configurations

Configure the low voltage warning setting, get and clear the low voltage warning flag status.

Low voltage warning configurations (optional)

Configure the high voltage warning setting, get and clear the low voltage warning flag status.

Configure the bandgap buffer (optional)

Function PMC_ConfigureBandgapBuffer configure the bandgap buffer working mode.

Peripherals and I/O pads isolation flag (optional)

Get/Clear the Peripherals and I/O pads isolation flag.
